## 시작 가이드

### Requirements

설치 요구 사항:

- [Node.js 20.10.0](https://nodejs.org/ca/blog/release/v20.10.0/)
- [Npm 10.2.3](https://www.npmjs.com/package/npm/v/10.2.3)

### Installation

```bash
$ git clone https://github.com/hyukjunkim1116/foodmap-Vue3
$ cd foodmap-Vue3
$ npm i
$ npm run dev
```
